# GT exercise

Tech stack:

- React + Typescript
- Vite
- Tailwind CSS
- Cypress

## Installation

```Bash
npm i
```

## Developing Locally

```Bash
npm run dev
```

## Buiding the App

```Bash
npm run build
```

### Testing the App Locally

```Bash
npm run build
npm run preview
```
